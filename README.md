GDork:Otomatize Dork Analiz Paneli 

GDork, kurumların dijital varlıklarını ve sızıntılarını tespit etmeyi amaçlayan gelişmiş bir Google Dorking otomasyon aracıdır.

Projenin Amacı
Bu araç, manuel olarak yapılması zaman alan Google Dorking süreçlerini otomatize ederek; siber güvenlik analistlerinin hedef kurumlar üzerindeki veri sızıntılarını (KVKK), açık dizinleri, yönetici panellerini ve yanlış yapılandırılmış sunucu dosyalarını saniyeler içinde tespit etmesini sağlar.

Öne Çıkan Özellikler

    Gelişmiş Dork Kütüphanesi: 40'tan fazla özgün ve kategorize edilmiş dork şablonu.

    Numarage Teknolojisi: Google'ın sayı aralığı (..) operatörünü kullanarak T.C. Kimlik No ve Telefon gibi hassas verilerin örüntü tabanlı tespiti.

    Modern Arayüz: Siber güvenlik temasına uygun "Cyber Dark" tasarım ve kullanıcı dostu "Dot Pagination" (Nokta Navigasyonu).

    Yüksek Performans: Go (Golang) dilinin sunduğu hız ve verimlilikle çalışan backend mimarisi.

Teknik Altyapı

    Backend: Go (Golang)

    Frontend: HTML5, CSS3, JavaScript

    Metodoloji: OSINT , Google Hacking

Kurulum ve Çalıştırma

    Projeyi bilgisayarınıza klonlayın:
    Bash

    git clone https://github.com/kullanici-adiniz/gdork.git

    Proje dizinine gidin:
    Bash

    cd GDork

    Uygulamayı çalıştırın:
    Bash

    go run .

    Tarayıcınızdan şu adrese gidin:
    http://localhost:5050
